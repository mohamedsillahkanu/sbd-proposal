🏢 About ICF-SL

Informatics Consultancy Firm-Sierra Leone (ICF-SL) is a leading technology consultancy in health informatics, mobile data collection systems, and digital health solutions across West Africa. Founded in 2022, ICF-SL specializes in delivering cutting-edge data-driven solutions to Non-Governmental Organizations (NGOs) and Government agencies.

Our Mission

We leverage cutting-edge technology to strengthen health systems and improve health outcomes in Sierra Leone and the broader region. We combine deep local knowledge with international best practices to deliver sustainable, scalable solutions.

Malaria Prevention Expertise

ICF-SL brings specialized expertise in malaria prevention programs, having supported the National Malaria Control Programme (NMCP) of Sierra Leone in deploying digital solutions during the 2023 ITN Mass Campaign. Key ICF-SL members supported the school-based ITN distribution pilot in Kono district in 2023. The firm is currently assisting in digitalizing malaria checklists for routine data quality assessments (mRDQAs), malaria onsite coaching and mentorship (MOCM).

3. Proposed Methods and Data Management

Technical Approach

A comprehensive mobile data collection system will be used (Clappia) as the primary platform, supplemented by custom dashboard development using Streamlit for real-time analytics and visualization.

Platform Selection Rationale:

  • Clappia: This is a Mobile App builder with offline capability, robust data validation, and enterprise-grade security
  • Streamlit: This is a Python-based framework for rapid dashboard development with real-time data visualization capabilities

Data Collection Tools Development

1. Logistics Tracking Form (Clappia App)

  • Real-time tracking of ITN stock distribution and delivery status
  • QR code scanner integration for inventory verification
  • GPS auto-capture for delivery point validation
  • Offline data entry with background sync upon connectivity
  • Automated notifications to stakeholders via Slack, SMS, and WhatsApp

2. ITN Distribution Tracking Form (Clappia App)

  • Dynamic form with conditional logic for student demographic capture
  • QR code scanner integration for rapid data entry
  • GPS auto-capture for school location verification
  • Offline functionality with automatic sync when online
  • Individual distribution records linked to enrollment database

3. Supervision and Quality Assurance Form (Clappia App)

  • Supervisor dashboard with check-in/check-out functionality
  • Digital checklist with photo documentation requirements
  • Issue reporting workflow with automatic escalation
  • Real-time communication with central team

Data Management Architecture

  • Mobile Platform: Clappia cloud infrastructure with enterprise security
  • Dashboard Platform: Python streamlit web application hosted on secure cloud server
  • Database: PostgreSQL database with automated backups
  • API Integration: RESTful APIs connecting Clappia data to Streamlit dashboard
  • Real-time Analytics: Python-based data processing with live dashboard updates
