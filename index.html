About ICF-SL

Informatics Consultancy Firm-Sierra Leone (ICF-SL) is a leading technology consultancy specializing in health informatics, mobile data collection systems, and digital health solutions across West Africa. Founded in 2018, we have successfully implemented over 50 health technology projects for international organizations including USAID, WHO, UNICEF, and local government agencies.

Our mission is to leverage cutting-edge technology to strengthen health systems and improve health outcomes in Sierra Leone and the broader region. We combine deep local knowledge with international best practices to deliver sustainable, scalable solutions that empower local communities and health workers.

ICF-SL has particular expertise in malaria prevention programs, having supported the National Malaria Control Program (NMCP) in implementing digital solutions for bed net distribution campaigns, malaria surveillance systems, and community health worker training programs across all 16 districts of Sierra Leone.

3. Proposed Methods and Data Management

Technical Approach

We propose implementing a comprehensive mobile data collection system using Clappia as the primary platform, supplemented by custom dashboard development using Streamlit for real-time analytics and visualization.

Platform Selection Rationale:

  • Clappia: No-code mobile app builder with offline capability, robust data validation, and enterprise-grade security
  • Streamlit: Python-based framework for rapid dashboard development with real-time data visualization and advanced analytics capabilities

Data Collection Tools Development

1. School Registration/Enrollment Form (Clappia App)

  • Dynamic form with conditional logic for student demographic capture
  • QR code scanner integration for rapid data entry
  • GPS auto-capture for school location verification
  • Photo capture capability with automatic cloud sync
  • Offline functionality with automatic sync when online

2. ITN Distribution Tracking Form (Clappia App)

  • Real-time inventory management with barcode scanning
  • Individual distribution records linked to enrollment database
  • Digital signature capture for verification
  • Automated timestamp and GPS logging
  • Push notifications for distribution milestones

3. Supervision and Quality Assurance Form (Clappia App)

  • Supervisor dashboard with check-in/check-out functionality
  • Digital checklist with photo documentation requirements
  • Issue reporting workflow with automatic escalation
  • Real-time communication with central team

Data Management Architecture

  • Mobile Platform: Clappia cloud infrastructure with enterprise security
  • Dashboard Platform: Streamlit web application hosted on secure cloud server
  • Database: PostgreSQL database with automated backups
  • API Integration: RESTful APIs connecting Clappia data to Streamlit dashboard
  • Real-time Analytics: Python-based data processing with live dashboard updates
